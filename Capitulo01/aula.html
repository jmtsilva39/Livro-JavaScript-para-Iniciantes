JavaScript é uma linguagem de tipagem dinâmica. Isso significa que não precisamos especificar o tipo de dado que uma variável vai armazenar.

Tipos de dados básicos em JavaScript

Number, String, Boolean, Null, Undefined, Object, Symbol.

Estrutura de Controle Condicionais

if, else if, else e switch

Estrutura de Controle: Loop

Loop são estruturas de controle que permitem repetir blocos de código até que uma condição específica seja atendida.

São elas: for, while e do while.

O loop do...while é semelhante ao loop while, porém a verificação da condição é feita depois da execução de um bloco de código, isto garante que o código seja executado pelo menos uma vez, mesmo que a condição seja falsa.
